Lets get started with some brief information. This page is a dedicated resource page for learning, and applying concepts and skills from Computer Science, Engineering, and Programming to both real life and educational problems. The first thing to do in preparation for the topics presented, and for future reference, is to have a table of programming/tech terms that will be mentioned frequently; the table assumes that those who are reading through this page have little to no programming knowledge. I have put a list in the resources section. Along with this list, I have added lists which both professional and educational resources.

List of Programming Terms

Term Definition
package a collection of libraries
library a collection of prewritten methods/functions that a program can use. Usually implemented by writing <include> or import followed by the library name
back-end the "back-end" of a program typically runs in the background; users do not interact with it
front-end the GUI of a program/application
GUI graphical user interface; a visible interface that is accessible/can be interacted with by the user
documentation the guides/information that are associated with a language, library, or code that someone has written; "docs"
Reached End of List
